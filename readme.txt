Hola, mi nombre es Marlon Santiago Henao, esta aplicación tiene como objetivo controlar un sistema IOT(cerradura inteligente), por medio
de sentencias HTTPS que son enviadas a un host, que tiene implantado un algoritmo escrito en PHP que es el encargado de enviar y recibir la información
hacia la base de Datos. Tiene una función de logueo para darle un toque de seguridad a la app, que es implementado con el authentication de
la API firebase de google.

servicios de host y base de datos ---> HOSTINGER
